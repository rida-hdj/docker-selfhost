# Self-Hosted Services

A collection of self-hosted Docker services running on a personal server.

> [!WARNING]  
> all of the variables are fake make sure you change it when you run any of this services

## Services

| Service | Description | URL |
|---------|-------------|-----|
| [Gitea](https://about.gitea.com) | Lightweight self-hosted Git service | `http://localhost:3000` |
| [Immich](https://immich.app) | Self-hosted photo and video backup solution | `http://localhost:2283` |
| [Jellyfin](https://jellyfin.org) | Free and open-source media server (movies, TV, music, photos) | `http://localhost:8096` |
| [Portainer](https://www.portainer.io) | Lightweight Docker container management UI | `http://localhost:9000` |
| [Samba](https://www.samba.org) | SMB/CIFS network file share | `smb://<host-ip>` (you can have acess to it with file manager) |
| [SearXNG](https://docs.searxng.org) | Privacy-respecting metasearch engine | `http://localhost:8080` |
| [Tududi](https://github.com/chrisvel/tududi) | Task and todo management application | `http://localhost:3002` |
